# Code of conduct

Flock's community should be useful, direct, and welcoming.

Treat other contributors with respect. Critique ideas and code rather than
people. Harassment, threats, discrimination, doxxing, and deliberately
disruptive behavior are not acceptable in project spaces.

Maintainers may edit or remove content and restrict participation when needed
to protect the community. Report conduct concerns privately through the
maintainers' GitHub profiles. Good-faith technical disagreement is welcome;
personal attacks are not.
